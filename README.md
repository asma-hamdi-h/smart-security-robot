#  Smart Security Robot (PFE Project)

An intelligent security robot system using IoT, Embedded Systems, and Artificial Intelligence.

---

##  Project Description

This project is a **smart security system** based on:
- ESP32 microcontroller
- ESP32-CAM for video surveillance
- Arduino programming
- Machine Learning / Artificial Intelligence
- Mobile application for remote control and monitoring

The system provides real-time monitoring, intrusion detection, and remote control through a mobile/web interface.

---

##  Features

-  Smart security monitoring
-  Real-time video streaming (ESP32-CAM)
-  AI-based detection (motion / intrusion)
-  IoT communication between devices
-  Mobile application control
-  User authentication system
-  Live dashboard visualization

## System Architecture

- ESP32 → controls sensors and actuators
- ESP32-CAM → captures video stream
- Backend server → processes data & AI detection
- Mobile App → user interface
- Cloud database → stores data
